Ubuntu Live USB Creator (usb-creator) is an official tool to create Live USBs
of Ubuntu from the Live CD or from an iso image. The tool is already included
in Ubuntu 8.10, but can also be installed on Ubuntu 8.04.

## [edit] Features

  * Install bootloader to USB device
  * Optional persistence (all changes made to the system are saved, unlike a Live CD)
  * Progress indicator
  * Does not format the device

## [edit] Future development

Currently the tool is only available for Ubuntu, but Windows and KDE versions
are planned. [1]

However, you can still install this utility through Kubuntu (or other official
derivataves, depending on what disc or ISO you select) with the command: `sudo
apt-get install usb-creator`
